== Description ==

The BrowserStack plugin allows you to open the current page in BrowserStack directly from the admin bar. No setup required.

BrowserStack gives you live, web-based browser testing with instant access to all desktop and mobile browsers.

This plugin borrows heavily from the BrowserStack integration on Codepen.io.

__What's next?__

* Remember your previous OS/browser choice.
* Support for web tunnels.

== Installation ==

This section describes how to install the plugin and get it working.

1. Add the `browserstack` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
